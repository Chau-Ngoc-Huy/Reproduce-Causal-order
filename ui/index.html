The published tables, beside ours

Every experimental table in the paper (Tables 1–6 and appendix Tables A3, A5–A13, A15–A17) is transcribed faithfully below. Wherever we reproduced a run, our GPT-4o value sits beside the paper's number, shaded by how it compares. The paper uses GPT-3.5 / GPT-4 (and Phi-3 / Llama3 / human annotators), so absolute values differ by model — the test is whether the method ordering and qualitative behaviour reproduce. We reproduced pairwise (CoT) + triplet on the loaded datasets; everything else is left blank.
