The benchmarks

Datasets in the source code

The repository defines 11 benchmark causal graphs in graphs/definitions.py — each a set of variables, a ground-truth DAG, optional descriptions, and a prompt context. This reproduction runs the ones marked Reproduced below; the rest are ready to run. Click any card for its full variable list.

Step by step

How the method works

Five conceptual stages — decompose / enumerate, ask the expert, vote / assemble, the recovered graph (predicted vs. ground truth, interactive), and the causal order. Pick a dataset and method, press play, or click a stage to jump. The final stage explains exactly how the causal order is computed.
